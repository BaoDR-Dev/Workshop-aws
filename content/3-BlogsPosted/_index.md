Here is a list and introduction of blogs that have been posted on [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - Guide to Migrating from Amazon EC2 to EKS Auto Mode with Kiro CLI and MCP Servers](3.1-Blog1/)

This blog guides how to migrate an application from EC2 to EKS Auto Mode using Kiro CLI and MCP Servers. The content covers why EKS Auto Mode is chosen to reduce infrastructure management overhead, how Kiro CLI analyzes and proposes a migration plan, how MCP Servers build validation gates for each step, and the actual process of containerizing, creating a cluster, and switching traffic to minimize risk.

### [Blog 2 - Automating the 3D Production Pipeline with AWS VAMS and 4D Pipeline](3.2-Blog2/)

This blog presents how AWS Visual Asset Management System (VAMS) combined with 4D Pipeline automates the entire 3D asset processing workflow from design in DCC tools like VStitcher to distribution on e-commerce platforms. The solution eliminates manual handoff steps, shortens time-to-market, and is applicable across many industries such as fashion, automotive, and furniture.

### [Blog 3 - AWS Launches Partner Agent Factory to Drive Enterprise AI Agents](3.3-Blog3/)

This blog introduces the Partner Agent Factory (PAF) program within the AWS Generative AI Innovation Center, where AWS collaborates with technology partners to build and validate enterprise-ready AI Agents. The post highlights real-world examples such as ETL Crew, Conversational Banking, and QAIssist, while analyzing why the Agentic AI trend is rapidly shifting from experimentation to production.
